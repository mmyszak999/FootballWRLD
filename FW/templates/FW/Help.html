Help

Create a club

Club name - type the club name you want to create ( real or fictional - whichever you want )

Foundation date - type the year of the club foundation (positive integer)

Country - type the native club country

Create a player

Name and surname - type the name and surname of the player in 2 fields (if player do not have a surname, leave surname field empty

Nationality - type the nationality of the player

Year of birth - type the year of birth of player (positive integer)

Club - click the drop-down list and pick one of the available clubs, which player will belong to

Position - click the drop-down list and pick the player's position

Height - type the height of the player in centimeters (positive integer)

Weight - type the weight of the player in kilograms (positive integer)

Foot - click the drop-down list and pick the player's better foot

Draw a random player

The site draws one player from the database. In order to draw another random player, you should press F5 button on your keyboard or click the refresh button in your browser

Filter players

Available filters:

Name - Filter a player/players with typed name (name can start with lowercase)

Name contains - Filter a player/players, whose name contains the typed phrase

Surname - Filter a player/players with typed surname (surname can start with lowercase)

Surname contains - Filter a player/players, whose surname contains the typed phrase

Nationality - Filter a player/players with typed nationality

Year of birth - Filter a player/players with typed year of birth value (positive integer)

Year of birth is less than - Filter a player/players, whose year of birth is lower than typed value (positive integer)

Year of birth is greater than - Filter a player/players, whose year of birth is greater than typed value (positive integer)

Club Filter a player/players, who belong to the club picked from the drop-down list

Position Filter a player/players, who play/s at the position indicated from the drop-down list

Height - Filter a player/players with typed height value (positive integer)

Height is less than - Filter a player/players, whose height is lower than typed value (positive integer)

Height is greater than - Filter a player/players, whose height is greater than typed value (positive integer)

Weight - Filter a player/players with typed weight value (positive integer)

Weight is less than - Filter a player/players, whose weight is lower than typed value (positive integer)

Weight is greater than - Filter a player/players, whose weight is greater than typed value (positive integer)

Foot - Filter a player/players, whose the better foot is the indicated one


You can get the filter results by clicking blue Filter button under the filters or by pressing Enter on your keyboard.
Filter results (if exist) appear as a table on the right side, after the filter run

Draft

In draft, there are 5 player slots. You can pick only 1 player from each slot. During the first four slots, you pick one player for one position (goalkeeper, defender, midfielder and attacker)

The last slot include the players, who were not picked before (except for goalkeepers - you do not need 2)

When you select the players, click blue button Build and you will see your team in the table

Clubs database and Players database

You do not need help here - it's just the table filled with clubs/players

If you click the club/player name, you will get some more info about it
